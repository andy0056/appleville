# Build Your Life in Himachal — Product Pack

This folder contains the initial product definition for the first version of the app.

## Files

- `PRD.md` — product requirements document
- `homepage-copy.md` — homepage copy draft
- `towns-v1.md` — town list and why each is included
- `quiz-scoring.md` — quiz questions and scoring logic
- `sitemap.md` — app structure / page map
- `name-ideas.md` — naming directions

## Product direction

Warm, earthy, local.

Usable product flow first.

Initial wedge:
- Help people figure out which town in Himachal fits their life and work needs.
